> Visual and interaction contract for Phase 2 of Galdi Gestión Tab 5 "Presupuestos".
> Generated by gsd-ui-researcher. Verified by gsd-ui-checker.
>
> **Governing constraint:** This phase adds two new cards (Card 6: Detalle and Card 7: Totales)
> inside the existing `#tab-presupuestos` panel. Every token and pattern MUST replicate those

## Interaction Contract

### Real-time recalculation (TOT-01, TOT-03)

Trigger: `oninput` event on `.pres-cant`, `.pres-precio`, `#pres-desc-valor`

Function `presRecalcLinea(inputEl)`:
1. Find parent `.pres-linea`
2. Read `.pres-cant` and `.pres-precio` values (default to 0 if empty/NaN)
3. Compute `rowTotal = cant × precio`
4. Write formatted value to sibling `.pres-subtotal-linea`
5. Call `presRecalcTotales()`

Function `presRecalcTotales()`:
1. Sum all `.pres-subtotal-linea` numeric values → `subtotal`
2. Read `#pres-desc-valor` and current type (pct or fijo)
3. If type = pct: `descuento = subtotal × (val / 100)`
4. If type = fijo: `descuento = val`
5. Clamp `descuento` to `[0, subtotal]`
6. `total = subtotal − descuento`
7. Update `#pres-val-subtotal`, `#pres-val-descuento`, `#pres-val-total`
8. Show/hide `#pres-row-descuento` based on whether `descuento > 0`

All values formatted as Chilean pesos: `'$' + Math.round(n).toLocaleString('es-CL')`

### Catalog dropdown cascade (LINE-01)

Function `presCargarProductos(catSelect)`:
1. Read selected category value
2. If empty: disable and reset `pres-prod-sel`, return
3. Filter in-memory product list (already loaded in Tab 1) by `categoria === value`
4. If Tab 1 product list not accessible: query Firestore `galdi_productos` where `categoria == value`
5. Populate `pres-prod-sel` with `<option value="{precioFinal}">{nombre}</option>`
6. Enable `pres-prod-sel`

Function `presAutocompletarPrecio(prodSelect)`:
1. Read `prodSelect.value` (contains `precioFinal`)
2. Set sibling `.pres-precio` value
3. Call `presRecalcLinea(prodSelect)`

### Add line (LINE-03)

Function `presAgregarLinea()`:
1. Generate unique ID: `Date.now().toString()`
2. Create new `.pres-linea` div from template (default mode: catálogo)
3. Append to `#pres-lineas`
4. Show `#pres-header-lineas`, hide `#pres-empty-lineas`
5. Focus the first select in the new row

### Delete line (LINE-04)

Function `presEliminarLinea(btn)`:
1. Find parent `.pres-linea`, call `.remove()`
2. If `#pres-lineas` is now empty: hide `#pres-header-lineas`, show `#pres-empty-lineas`
3. Call `presRecalcTotales()`
4. No confirmation dialog — deletion is immediately reversible by adding a new line

### Discount type toggle (TOT-02)

Function `presSetDescTipo(tipo)`:
1. Update `#pres-desc-tipo-pct` and `#pres-desc-tipo-fijo` active styles
2. Update `#pres-desc-simbolo` text: "%" or "$"
3. Reset `#pres-desc-valor` to 0
4. Call `presRecalcTotales()`

Active button style: `background: var(--dark); color: var(--gold); font-weight: 600`
Inactive button style: `background: #fff; color: var(--muted); font-weight: 400`

## Number Formatting

All monetary values throughout Cards 6 and 7 use Chilean peso format:
- No decimal places (pastries are priced in whole pesos)
- Dollar sign prefix: `$`
- Thousands separator: dot (Chilean locale `es-CL`)
- Implementation: `'$' + Math.round(n).toLocaleString('es-CL')`
- Examples: `$0`, `$1.800`, `$25.000`, `$1.200.000`

Quantity field: integer only (`type="number"`, `min="1"`, `step="1"`).
Unit price field: integer only (`type="number"`, `min="0"`, `step="1"`).
Discount value (pct): decimal allowed (`type="number"`, `min="0"`, `max="100"`, `step="0.1"`).
Discount value (fixed): integer (`type="number"`, `min="0"`, `step="1"`).

## Responsive Behavior

Inherits Phase 1 breakpoint: `@media(max-width: 600px)`.

At 600px and below, `.pres-linea` and `#pres-header-lineas` switch to:
```css
@media(max-width: 600px) {
  .pres-linea, #pres-header-lineas {
    grid-template-columns: 1fr 1fr;
    grid-template-rows: auto auto auto;
  }
}
```
Simplified mobile layout (2-column grid, 3 rows per line):
- Row 1: mode toggle (col 1) + description (col 2)
- Row 2: quantity (col 1) + unit price (col 2)
- Row 3: row subtotal (col 1) + delete button (col 2)

Column headers hide on mobile (`display: none` for `#pres-header-lineas`).
Totales card: discount toggle and input stack vertically (`flex-wrap: wrap` already on container).
